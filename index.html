Dappcord

Dappcord is a decentralized chat application built on the Ethereum blockchain. As a developer on the project, I worked with a range of cutting-edge technologies to deliver a seamless user experience. The backend of Dappcord was built using Solidity. I used the Hardhat development environment to write, test, and deploy our smart contracts. Hardhat provided a suite of tools for testing and debugging, which helped us to ensure that our contracts were secure and functional. To interact with our smart contracts, I used Ether.js, a JavaScript library that allows developers to easily interface with the Ethereum network.
